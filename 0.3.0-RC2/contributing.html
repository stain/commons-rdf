Contribute

Contributions to this project will be gratefully received.

Feel free to subscribe to the dev@commonsrdf mailing list to follow the ongoing development of Commons RDF, ask questions about its usage, or help shape Commons RDF by contributing your ideas, code and use cases.

You can contribute to Apache Commons RDF in many ways:

  • Participate in the design discussions on dev@commonsrdf
  • Implement the Commons RDF API in your own implementation
  • Raise an issue
  • Raise a pull request to:
    • Contribute new tests
    • Suggest changes to the API
    • Improve the Javadocs
    • Fix bugs in the simple implementation
    • .. and more
  • Improve the website by:
    • Raising a pull request for changes to src/site
    • Contribute example use cases

Contributions are assumed to be under the Apache License, version 2.0, however we may ask you to also sign a software grant for any larger contributions.
